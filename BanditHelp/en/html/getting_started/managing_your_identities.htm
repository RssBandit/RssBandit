Managing Your Identities

As of version 1.3, RSS Bandit provides support for managing multiple identities. This is especially suited for those suffering from multiple personalities disorder.

This feature addresses the fact that subscribing to RSS feeds is not merely a passive activity. Many RSS readers are actively involved with feeds that support the ability for users to post comments.

Some feeds request personal information from users who wish to post a comment. Depending on how much you trust a given feed, you may wish to control the information you supply. To give you a handle on this, RSS Bandit provides the option to create one or more identities and then choose an identity when posting a comment.

To see this feature in action, click on the Tools menu item. You'll see the Manage Identities menu option as in Figure 1.


Figure 1. Manage Identities Menu Option.

Selecting this option will open the dialog in figure 2.


Figure 2. Manage Identities Dialog.

In the left pane is a list of identities. Figure 2 shows two separate identities used by the author. The right pane displays the details for the selected identity. To add an identity, click the Add Identity button on the toolbar. As of version 1.3, the News Server options do not work.

When posting a comment to a website, the post dialog allows you to select an identity to use. For more information on posting comments, see Posting Comments.
