Deleting Feed Items

Deleting Individual Items
Starting with version 1.3, RSS Bandit allows deleting individual feed items. To delete an item, simply select the item and press the "Delete" key. You may also right click the item and select "Delete" on the context menu.

Deleting an item places it in the new Special Feed named "Deleted Items" as in Figure 1. Items remain here until you empty the bin by right clicking it and selecting "Delete Items".


Figure 1: Closeup of the Deleted Items Recycling Bin.

Automatically Deleting Old Items
RSS Bandit allows the user to set an option to automatically delete posts older than a specified number of days. For example, RSS Bandit can be set to delete all posts older than 30 days.

There are two places where this setting applies. One is a global setting that applies to all subscribed feeds. Likewise, individual feeds can be configured to override the global setting on a per feed basis.

Global Settings
To access the global setting, click on the Tools menu and select Options.... In the Options dialog box, click on the Feeds tab (see Figure 2). The setting "Default Maximum Item Age:" is the global setting for when to delete a post.


Figure 2: Configuring the global setting for when to delete feed items.

Per Feed Setting
To access this setting for a particular feed, right click on the feed and click Properties.... In the resulting dialog, the Max. Item Age drop down allows you to select a value for when items within the feed should be automatically deleted (See Figure 3).
