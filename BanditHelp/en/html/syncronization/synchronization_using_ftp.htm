Synchronization using FTP

Selecting FTP allows you to remotely store your feeds on an FTP server. Figure 1 displays the configuration settings for configuring an FTP server.


Figure 1: Remote Feeds Dialogue - FTP

The first field is the FTP server/path. This is the IP address or Domain name of the FTP server followed by the path to the folder in which to store the synchronization data. Figure 1 shows the case where the feed data is stored in the folder feedlistfolder of the ftp site ftp://yourserver.com.

The next two fields are your User Name and Password. If your FTP site is hosted by an ISP, they should have provided you this information. RSS Bandit will use these fields in order to login to your FTP site and upload/download feed data.
