RSS Bandit’s local search feature is powered by the DotLucene search engine, a direct port of the Apache Lucene engine for Java.

There is no need to know this fact for simple searches. Just type in a search term and go. But to realize the full power of the Lucene search engine, it helps to learn the Lucene Query Syntax.

Searchable fields

For more advanced searches you can use the following fields (mentioned in the lucene sytax docs) to find terms e.g. a feed description or feed title:

author
Item Author. Not supported by all feeds.
title
Item Title, aka. headline.
link
Item permanent Link.
date
Date the Item was posted. The index date resolution is minute. So a valid date search expression looks like this: date:20061224* will find all items posted at December, 24th 2006, and date:200612241300 will find the item(s) posted exactly at December, 24th 2006, 1:00 PM. Please note: the displayed dates in the item list and detail pane show the item dates transformed to your local system timezone!
topic
Item Topic, aka. category as provided by the feed publisher.
feedlink
Feed Link. The feed subscription url itself.
feedurl
Feed Url. This is in common the homepage of the feed publisher.
feedtitle
Feed title as provided by the feed publisher.
feeddescription
Feed description as provided by the feed publisher.
feedtype
Type of the feed: 'RSS' or 'NNTP'
