Newspaper Views

As of version 1.3, RSS Bandit supports FeedDemon compatible Newspaper Views. This is a convenient method for reading multiple feed items at a time within the reading pane. It is especially suited for News feeds.

When clicking on a category or a feed in the Feed List, every unread item is displayed within the reading pane. Figure 1 below shows an example in which a category containing feeds from technology oriented news sites is selected. This example demonstrates the "Headlines" newspaper style.


Figure 1. Newspaper view of a category using the Headlines style.

Figure 2 is an example of a single feed using the "Film" newspaper style.


Figure 2. Newspaper view of a single feed using the Film style.

Configuring Display Style

The style (aka formatter) used to display a feed can be configured globally, per category, and per feed.

Global Style Setting
The global setting is in the Tools | Options dialog under the "Display" tab (see figure 3).


Figure 3. Configuring the global (default) style.

By checking "Use a custom formatter", you can select one of the available stylesheets. The style configured in this manner is used for any feed or categories in which a custom formatter is not specified.

Category Style Setting
Right clicking on a category folder selecting "Properties..." brings up the dialog as shown in figure 4. Click on the "Category" tab to configure the Category style.


Figure 4. Configuring the category style.

Feed Style Setting
Finally, by right-clicking on a feed and bringing up the properties dialog, clicking the "Display" tab will give you the dialog shown in figure 5. This allows you to customize a formatter per site.


Figure 5. Configuring a feed's style.

You'll notice there's an option to "Mark items as read on exiting category". Checking that can improve the Newspaper viewing experience as all items in the category are marked as read upon moving to the next category.
