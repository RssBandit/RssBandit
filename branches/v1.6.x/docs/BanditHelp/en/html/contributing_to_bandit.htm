Contributing to RSS Bandit

To contribute to the project you can choose whatever section of work you like. All sources including this documentation can be found at sourceforge.net. You should be somewhat familar with CVS and need an account at sourceforge.net (we prefer to know who works so hard to contribute).

Start contributing by providing patches at http://sourceforge.net/tracker/?group_id=96589&atid=615250. They should be "fresh", mean nearly the current state/version of the sources from within the CVS. Please describe what the patch is for and provide diff file(s).
