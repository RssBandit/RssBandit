Using a file share simply copies the file to a network file share specified in the UNC directory path. Figure 1 shows an example where RSS Bandit is configured to store the Feeds data in the RSSBanditFeeds folder of the server YourServer.


Figure 1: Remote Feeds Dialogue - File Share

Obviously, you will need read/write access to this network share from each computer with which you wish to synchronize RSS Bandit.
