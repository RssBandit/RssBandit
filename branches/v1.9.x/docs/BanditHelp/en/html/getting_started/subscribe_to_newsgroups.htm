Subscribe to Newsgroups

Along with subscribing to feeds, RSS Bandit supports subscribing to USENET newsgroups via NNTP. Newsgroups are a natural complement to subscribing to blogs.

The first step in subscribing to a newsgroup is to configure a news server. Under the Tools menu select the Newsgroups... menu item. This will bring up the Manage Newsgroups dialog as in figure 1.


Figure 1. Manage Newsgroups Dialog.

Click on the Add News Server button to add a newsgroup server. A news server is required to connect to the vast number of newsgroups out there. There are many free choices. For the purposes of this documentation, I chose news.edisontel.com as you can see in figure 2.


Figure 2. News Server General Settings.

In the field labeled News Account enter in a friendly name for the news server. For the Default Identity select an identity to associate with your newsgroup postings. If you don’t have an identity in the drop down you can set one up.

Click on the Server Settings option in the left treeview to enter the server connection information. In the Server Name field enter the DNS entry (or IP address) of the news server as in figure 3.


Figure 3. News Server Settings.

If you need to change one of the advanced settings for the server, click on the Advanced Settings option in the tree view. This allows you to configure the Server Port Number, whether or not it requires SSL, and a timeout as in figure 4.


Figure 4. News Server Advanced Settings.

To display a list of all the newsgroups on the server, click on the news server within the tree view. Then click the Refresh List button. After a brief moment, the list view will be populated with the available newsgroups as in figure 5.


Figure 5. News Group List.

To find a particular newsgroup, you can use the text box which filters the list. For example, by typing microsoft.public.dotnet.framework the list is filtered to all newsgroups that match that string as in figure 6.


Figure 6. News Group Filter.

To subscribe to a newsgroup, either double click the newsgroup in the list or select the newsgroup and click the Subscribe button. This will bring up the Add New Subscription dialog. This dialog is alternatively reachable from the RSS Bandit GUI by clicking on the New... button or via the File menu.

At this point, you can simply follow the steps in the Add Subscription wizard as described in the topic Subscribing to a Feed. Once you complete the wizard, the newsgroup will show up in your list of feed subscriptions. Figure 7 shows a view of RSS Bandit with a subscription to a Microsoft newsgroup. It also shows a threaded newsgroup conversation.


Figure 7. Newsgroup subscription.

To post a reply to a newsgroup item, right click the entry and click Post Reply. This works in the same manner as posting a comment to a blog with the CommentAPI enabled.
