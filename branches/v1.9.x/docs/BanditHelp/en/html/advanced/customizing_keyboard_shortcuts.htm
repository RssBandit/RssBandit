Customizing Keyboard Shortcuts

RSS Bandit now allows customizing keyboard shortcuts with a straightforward dialog. To get started, right click anywhere on the tool bar and click Customize... in the context menu.


Figure 1: Customize Context Menu.

This brings up the toolbar customization dialog.


Figure 2: Toolbar Customization Dialog.

At the bottom of the dialog is a button to bring up Keyboard options. Click it to configure keyboard settings.


Figure 3: Keyboard Settings Dialog.

Keyboard settings are grouped into categories as shown in the left list. In the right are the list of commands. Once you find the command you wish to configure, just select a key combination in the dropdown in labeled Specify a Shortcut and click Assign.

You can always reset the settings back to their defaults by clicking Reset All...
