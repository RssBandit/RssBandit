Help on Help

To close this help window,use the button on the very top right. To close the Navigation component to get more space to read a topic, click the button.

At any time you are able to make the Navigation component become apparent by usage of the toolbar on top of this window. There are these tools available:

  • Contents. This tool displays a tree structure of the whole bunch of help topics available here and works similar to the well known Windows File Explorer. The Sync button can be used to synchronize the current displayed topic with the corresponding entry within the table of contents if the topic is not yet highlighted.

  • Index. With help of the index you may search very fast for specific key words.

  • Search. If the index is not sufficient to get the topic you want, then use the search capability. This will apply a full text search with the provided search expression.

  • Glossary. This is a collection of important terms stored together with their explanations.

  • The "Search" input field. With help of that field you are able to apply a full text search irrespective of the selection within the Navigation component. A click to the [go] button will then start the search process.
