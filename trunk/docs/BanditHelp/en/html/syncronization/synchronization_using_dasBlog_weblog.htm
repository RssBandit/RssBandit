Bandit Topic

DasBlog is an open source blogging engine. If you are a DasBlog or DasBlog 1.3 user, you can use its storage web service. Figure 1 shows an example of configuring DasBlog remote storage.


Figure 1: Remote Feeds Dialogue - DasBlog

The Web Service URL should be something along the lines of http://yourblog.com/dasblog/ConfigEditingService.asmx. Enter your user name and password into the User Name and Password fields.

For more information about DasBlog visit http://www.dasblog.net/.
