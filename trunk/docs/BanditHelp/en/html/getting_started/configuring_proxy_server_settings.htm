Configuring proxy server settings

A proxy server (or proxy for short) is basically the internet equivalent of a "middle-man" between you and the Internet. Typically, it allows a you to access the Internet from within a firewall. If you currently are having no problems subscribing to online feeds with Rss Bandit, you may not need to fiddle with the proxy server settings.

In order for a client application (such as Rss Bandit!) to access the web via, the client must be configured to access the web through the proxy server. In order to configure Rss Bandit's proxy settings, go to the Tools menu and select Options to bring up the Preferences dialog. Click on the Network Connections tab to bring up the dialog you see in Figure 1.


Figure 1. The Network Connection dialog.

By default, the item Don't use a proxy server is checked. This is the correct setting if you have a direct connection to the internet.

Internet Explorer Settings
If you are currently experiencing connectivity problems with Rss Bandit, but you have no problem exploring the internet with Internet Explorer, try the second setting Take over proxy server settings from Internet Explorer. Rss Bandit will use the same settings that Internet Explorer is using.

Private Proxy Settings
Finally, if you are still having problems, check the Use private proxy server settings options. This will allow you to manually enter in the IP address (or DNS host name) for the proxy server and a port number. If you do not know this information, please contact you Internet Service Provider (ISP) or your System Administrator.

The setting Bypass proxy addresses for local addresses is checked by default and you'll most likely want to keep it this way. However, in the rare situation that you are trying to read an rss feed from an internal server with a proxy in between, you may uncheck this option.

To exclude other servers from the proxy, enter a list of addresses separated by semicolons.

Authentication
If your proxy server required authentication, check the Proxy needs authorization checkbox and enter in your User Name and Password. Again, if you don't know this information, contact your ISP or System Administrator.
