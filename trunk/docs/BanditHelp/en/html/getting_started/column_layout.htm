Column Layout

As of version 1.3, RSS Bandit provides more control over which columns are displayed within the headlines section, as well as the order in which they are displayed.

Reordering columns
To reorder columns, simply drag and drop the columns in the order you wish. For example, figure 1 shows a four column headlines view. To move the "Comments" column to the right of the "Topic" column, simply click and hold the column and drag it over as in Figure 1 and drop it into place.


Figure 1. A column re-ordering in progress.

Figure 2 shows the result of the ordering.


Figure 2. After the reordering.

Selecting Columns To Display
You can also select which columns you wish to display within the headlines section. These Column Layout settings can be configured on the global, category, and feed level.

Global To select the columns to display on a global scale, select Column Layout from the View menu as in figure 3.


Figure 2. Global Column Layout Menu.

The columns with a check mark are the ones selected for display. The menu also provides options to use this layout as the default for categories and/or for feeds. The final option is to reset the layout to the default layout that comes with RSS Bandit.

Feed / Category Changing the column layout for a feed or category is handled in the same manner as the global setting. For example, right clicking a feed and selecting "Column Layout" will display the context menu in figure 3.


Figure 3. Feed/Category Column Layout.

As with the global setting, the checked columns are displayed. Selecting "Reset layout to default" will the displayed columns to the default setting. In this scenario, the default setting may be different than the setting installed with RSS Bandit depending on whether or not you've selected "Use layout as default for feeds" at some point.
