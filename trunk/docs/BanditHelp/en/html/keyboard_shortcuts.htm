Keyboard Shortcuts

Viewing and exploring feeds

To do this Press this
Refresh all feeds subscribed to Ctrl-F5
Refresh the specific selected feed (focus on the treeview or listview) F5
Catch Up the selected feed, mark all the feed items read Ctrl-Q
Select all feed items in the listview Ctrl-A
Mark all selected feed items unread Ctrl-U
Rename a category or feed title within the tree F2
Subscribe to a new feed, create a new feed F4
Create a new category, treeview focused Shift-F4
Delete a feed or category, treeview focused Del
Autodiscover feeds Ctrl-S
Post a reply to the selected feed item Ctrl-R
Move to the next unread feed item (focus on the treeview or listview) Space
Navigate between the treeview, listview and item detail pane Tab, Shift-Tab
Switch between Feed Details Tab and the various Browser Tabs Ctrl-Tab, Shift-Ctrl-Tab
Close the current Browser Tab Ctrl-F4
Show Feed Properites Ctrl-P
Show the Options dialogue Ctrl-O
Download Feeds Ctrl-Shift-D
Upload Feeds Ctrl-Shift-U
Close and Exit Alt-F4
Import Feeds Ctrl-I
Export Feeds Ctrl-E
View Help F1
Update Category Shift-F5

Viewing and exploring web pages

To do this Press this
Open a new Browser Tab Ctrl-N
Refresh the Browser Tab Content (should have the focus) F5
Navigate back in the history of the current Browser Tab Alt-Left
Navigate forward in the history of the current Browser Tab Alt-Right
Set the focus to/select the Url navigation dropdown F11
Set the focus to/select the Search phrase dropdown F12
