Sometimes items don't appear for a feed even though I know there are items in the RSS feed

A: RSS Bandit is configured to show you posts for a specific time span. By default this value is one month. This means that posts older than month do not show up in the list of items for that feed. To change this number either go to the Tools->Options menu and select the Feed tab where you can change the 'Default maximum item age'. You can also change this on a per feed basis by left clicking on a feed in the tree view and choosing the Properties menu option.

I want to stop a feed from refreshing for a while, but I do not want to delete the feed from my subscriptions because I do not want to loose the link. Is it possible?

A: Yes. Select the desired feed and open the properties dialog. Then set the refresh rate to 0 (zero). Such feeds are visualized with a greyed out icon.

While reading the feed items I often click to the 'Read On...' link provided. It always selects the first web tab. But I want to open it in a separate tab!

A: This behavior is by design to do not waste resources (embedded Internet Explorer) and prevent too much opened tabs a user cannot remember. But you can use <Ctrl-Click> to the link to open this specific one on a new tab, or use <Shift-Click> to open a separate Internet Explorer window or a new separate Tab. This behavior can be configured within the Tools|Options|Web Browser dialog.

How do I delete feed items?

A: You can delete a feed item by selecting it and pressing the "Delete" key. You can also set RSS Bandit to automatically delete items that are older than a specified number of days. Please see the following help topic: Deleting Feed Items.
